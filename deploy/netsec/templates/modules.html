Agent 能力说明

Agent 客户端运行在目标机器上，通过 WebSocket 连接云端大脑，可以：

  • 执行系统命令（白名单安全限制）
  • 运行扫描工具（nmap、dirb 等）
  • 读取文件内容、获取系统信息
  • 模拟键盘鼠标操作
  • 截取屏幕截图
启动 Agent 客户端： 
# 在目标机器上运行
cd AI_Assistant
python agent_client.py

# 或设置环境变量
set BRAIN_URL=http://你的服务器IP:5000
python agent_client.py
