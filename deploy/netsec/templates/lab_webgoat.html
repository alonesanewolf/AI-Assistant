关于 WebGoat： WebGoat 是 OWASP 维护的一个故意不安全的 Web 应用程序，旨在教授 Web 安全课程。 它的设计理念是"边学边做"，每个漏洞都配有详细的课程说明和交互式练习， 是 Web 安全入门的经典学习平台。

推荐学习路径
入门阶段

从 HTTP 基础、代理配置开始，学习使用 Burp Suite/ZAP 抓包。

基础攻击

学习 SQL 注入、XSS、CSRF 等常见 Web 漏洞的原理和利用。

进阶攻击

挑战访问控制绕过、加密缺陷、反序列化等高级漏洞。

防御实践

学习安全编码规范、输入验证、输出编码等防御手段。

环境信息
版本：WebGoat 8.x
技术栈：Java + Spring Boot
运行方式：Docker / JAR 包
默认端口：8080 (WebGoat) + 9090 (WebWolf)
辅助工具：WebWolf（内建攻击辅助工具）
安全提示
  • WebGoat 自带课程提示和解决方案
  • 推荐配合 Burp Suite / ZAP 学习
  • WebWolf 可辅助完成部分课程
  • 课程技术不可用于未授权目标
  • 遵守法律法规和道德规范
